##  Entrega 1: Relatório de Escopo e Planejamento

### 1. Identificação do Projeto
**Tema:** Saúde Pública (Imunização de Rotina).
**Título:** Monitor de Cobertura Vacinal e Hiatos Imunológicos Municipais.
**Sociedade Impactada:** Gestores de saúde pública municipal, secretarias de saúde e organizações que atuam na vigilância epidemiológica.

### 2. Escopo do Projeto 
Desenvolvimento de uma **Aplicação Web** voltada para a Ciência de Dados que permita a visualização de indicadores e KPIs sobre a vacinação de rotina no Brasil. O sistema transformará dados brutos em inteligência geográfica para identificar áreas com baixa cobertura vacinal.

**Funcionalidades Principais:**
**Dashboards Interativos:** Visualização de metas vs. alcance real por vacina.
**Análise de Tendência:** Comparativo anual para detectar queda na imunização.
**KPIs de Desempenho:** Indicadores de abandono vacinal e taxa de cobertura por município.

3. Justificativa e Metodologia 
O tema foi selecionado por sua relevância social e pela viabilidade de dados. Diferente de temas sazonais (como Dengue), a vacinação de rotina exige monitoramento constante. Utilizaremos dados públicos do DATASUS (TABNET) e Kaggle
**Metodologia de Trabalho:**
1.  **Coleta e Limpeza:** Extração de dados via Python e tratamento com a biblioteca Pandas.
2.  **Análise de Dados:** Aplicação de métricas para identificar "bolsões de baixa imunidade".
3.  **Desenvolvimento Web:** Criação de painéis e relatórios para suporte à tomada de decisão.
4.  **Versionamento:** Todo o ciclo de vida do código será gerenciado via GitHub.
